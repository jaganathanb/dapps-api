# Golang Clean Web API

## Used Tools

1. [Gin as web framework](https://github.com/gin-gonic/gin)
2. [JWT for authentication and authorization](https://github.com/golang-jwt/jwt)
3. [Redis for caching](https://github.com/redis/redis)
4. [Elasticsearch for logging database](https://github.com/elastic/elasticsearch)
5. [Beat for log shipping](https://github.com/elastic/beats)
6. [Kibana as log viewer](https://github.com/elastic/kibana)
7. [Prometheus for metric database](https://github.com/prometheus/prometheus)
8. [Grafana for metric dashboards](https://github.com/grafana/grafana)
9. [Validator for endpoint input Validation](https://github.com/go-playground/validator)
10. [Viper for configurations](https://github.com/spf13/viper)
11. [Zap for logging](https://github.com/uber-go/zap)
12. [Zerolog for logging](https://github.com/rs/zerolog)
13. [Gorm as ORM](https://github.com/go-gorm/gorm)
14. [Swagger for documentation](https://github.com/swaggo/swag)
